# CareerCompass-AI 🎓

An AI-powered hybrid recommendation system that analyzes university students'
academic performance, skills, and interests to suggest personalized career paths
and relevant courses.

## Tech Stack
- Python, LightFM, scikit-learn, XGBoost, NLTK, TF-IDF

## Project Structure
- `data/` — datasets used for training
- `notebooks/` — Jupyter notebooks with model training
- `models/` — saved trained models
- `README.md` — project overview
